# RCS Ollama

More information: https://carleton.ca/rcs/llm-access-instructions/ 

LLM Server: https://rcsllm.carleton.ca/rcsapi

Requirements: 
- Ensure that you have connected to the VPN
- Ensure that you have an API key

If you have any more questions, ask Neel

## Configuring/Running the Demo

Create a venv and install the requirement (only requirement is ollama)

```
# CREATE VENV FIRST
pip install ollama
```

Run main.py, passing your API key as an argument

```
python main.py --api_key <your api key>
```
